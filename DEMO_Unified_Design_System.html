2. TYPOGRAPHY

Font Families

Primary Font: 'Segoe UI', 'Calibri', 'Arial', sans-serif
Monospace Font: 'Courier New', monospace

Font Sizes & Weights

Element Size Weight Example
H1 (Main Title) 18px bold Main Title
H2 (Section) 14px bold Section Title
H3 (Subsection) 12px bold Subsection
Body Text 11px normal Regular paragraph text
Small Text 10px normal Captions and footnotes

Line Height

  • Body text: 1.6 (spacious, readable)
  • Tables: 1.5 (compact for data)
  • Forms: 1.6 (comfortable spacing)
3. SPACING SYSTEM

All spacing uses a consistent 4px baseline grid:

Variable Size Use Case
--spacing-xs 4px Minimal spacing
--spacing-sm 8px Small gaps, list items
--spacing-md 15px Standard paragraph spacing
--spacing-lg 20px Section margins
--spacing-xl 30px Major section breaks
--spacing-xxl 40px Signature blocks, document breaks

8. DESIGN SYSTEM SUMMARY
Unified Design Across All Templates

All 46 DPDP compliance templates now use this consistent design system:
  • Consistent Colors: Primary blue (#0066cc), dark headers (#2c3e50), status colors
  • Consistent Typography: Segoe UI/Calibri fonts, 11px body, 14px headers
  • Consistent Spacing: 4px baseline grid with named spacing variables
  • Consistent Components: Tables, forms, info boxes, signatures
  • Professional Appearance: Suitable for board presentation and client delivery
  • Print-Ready: All templates print cleanly to PDF

Stylesheet Reference

All templates link to a single CSS file:
<link rel="stylesheet" href="UNIFIED_THEME.css">

This ensures consistent styling across all templates and makes future updates centralized and easy to manage.
