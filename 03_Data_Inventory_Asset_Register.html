Data Inventory & Asset Register
Complete inventory of all systems and data stores holding personal data
Purpose: Identify and document every system, database, file store, or application that holds personal data. This is the foundation for ROPA (Records of Processing Activities) mapping.

Instructions

  1. List every system, application, database, spreadsheet, or cloud service that stores personal data
  2. Include systems run by your organization AND those run by processors on your behalf
  3. For each system, document: ownership, data types, retention period, location, security classification
  4. Review with IT, Business, and Legal teams to ensure completeness
  5. Update this inventory whenever new systems are added or data handling changes
