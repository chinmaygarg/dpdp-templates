## 📖 How to Customize

Each template includes:
1. **Header** with version and effective date
2. **Overview** section explaining purpose
3. **Content sections** ready for customization
4. **Sign-off** area for approvals
5. **Footer** with metadata

**To customize:**
1. Open template in browser
2. Replace `[Organization Name]` with actual name
3. Replace `[DATE]` with effective dates
4. Fill in tables and forms
5. Print to PDF or save as HTML
6. Obtain sign-offs (Legal, DPO, etc.)
